##web服务案例
####应用层
- 只解决要传递什么数据
- ftp
- ssh
- smtp
- pop3
- HTTP
    - 超文本传输协议
    - 请求方式
        1. GET  获取数据
        2. POST  修改数据
        3. PUT  保存数据
        4. DELETE  删除
        5. OPTION  询问服务器的某种支持特性
        6. HEAD  返回报文头
